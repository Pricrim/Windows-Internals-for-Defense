cmd.exe

The Windows Command Processor (cmd.exe) is one of the most common administrative tools in Microsoft Windows environments.

  • Location: C:\Windows\System32\cmd.exe
  • Type: Command Interpreter
  • ATT&CK: T1059.003
  • Category: Execution
